# Project Structure

## specs/

- `BOARD.md` - operational board.
- `WAL.md` - short session memory.
- `ROADMAP.md` - implementation waves.
- `TECHDEBT.md` - accepted compromises.
- `protocols/*` - local workflow rules.

## specs/common/

- `main.md` - project intent.
- `structure.md` - this file.
- `PROP-000-base-rules.md` - base spec-driven rules.

## specs/modules/core/

- `PROP-*` - cross-feature product and domain canon.
- `FEAT-*` - user-visible behavior.
- `INFRA-*` - Windows runtime, install, permissions, and operational concerns.

## Future source layout

- `src/` - implementation.
- `tests/` - verification.
- `tools/` - install/dev helper scripts.
