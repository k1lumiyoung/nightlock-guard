# WAL-PROTOCOL

`WAL.md` is short session memory.

It contains:
- active sessions;
- completed work;
- cross-module notes;
- decisions pending.

Active sections use:

```md
### FEAT-001: Title (@handle)
- Started: YYYY-MM-DD
- Spec: `spec://modules/core/FEAT-001-title#root`
- Status: In Progress
- Summary: Current short handoff state.
```

Keep `WAL` brief. Long reasoning belongs in specs, code review, or commit history.
